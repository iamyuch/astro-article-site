# Astro静态网站生成器入门指南

Astro是一个现代化的静态网站生成器，专注于构建快速、内容驱动的网站。它采用独特的"群岛架构"（Islands Architecture），允许开发者在需要时才加载JavaScript，从而实现极致的性能优化。

## 核心特性

### 零JavaScript默认输出

Astro默认生成零JavaScript的HTML页面，这意味着您的网站加载速度极快。只有在需要交互功能时，才会选择性地加载JavaScript。

### 支持多种框架

您可以在同一个项目中使用React、Vue、Svelte等多种前端框架。Astro会自动处理这些框架的集成，让您可以选择最适合每个组件的工具。

### 内容集合

Astro提供了强大的内容集合API，可以轻松管理Markdown和MDX文件。这使得构建博客、文档站点等内容驱动的网站变得非常简单。

## 快速开始

安装Astro非常简单，只需运行以下命令：

```bash
npm create astro@latest
```

然后按照提示选择模板和配置选项即可。

## 总结

Astro是构建静态网站的优秀选择，特别适合内容驱动的项目。其性能优势和灵活性使其成为现代Web开发的有力工具。
